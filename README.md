# MyBooks

A modern book management application built with Laravel, Inertia.js, and React.

## Features
- User authentication
- Complete CRUD operations for books
- Book status management (Read/Unread)
- Star rating system
- Categories and genres
- Search and filtering
- Image upload for book covers
- Statistics dashboard

## Tech Stack
- Laravel 10+
- Inertia.js
- React
- Tailwind CSS
- MySQL
